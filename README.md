# 🌾 KrishiSetu
### Smart Agriculture Assistance and Farmer Support Platform

KrishiSetu is a smart agriculture web platform developed to help farmers with modern digital farming solutions.  
The platform provides crop recommendation, disease detection, weather updates, government schemes, expert guidance, and agricultural articles in one place.

---

# 🚀 Features

✅ Farmer Registration & Login  
✅ Farmer Profile Management  
✅ Crop Recommendation System  
✅ Leaf Disease Detection  
✅ Weather Forecast by Location  
✅ Government Schemes Section  
✅ Agricultural Articles & Search  
✅ FAQ Section  
✅ Expert Query Support  
✅ Responsive Dashboard UI

# ⚙ Installation Steps

```bash
# Clone repository
git clone <your-github-repo-link>

# Open project folder
cd KrishiSetu

# Create virtual environment
python -m venv venv

# Activate environment
venv\Scripts\activate

# Install requirements
pip install -r requirements.txt

# Run migrations
python manage.py makemigrations
python manage.py migrate

# Start server
python manage.py runserver
